# Bake & Grill - Color Scheme Documentation

## Current Color Scheme

The Bake & Grill website currently uses a **green and gold color palette** that reflects the authentic Dhivehi cuisine theme.

### Current Colors
```css
:root {
    --primary-color: #059669;        /* emerald-600 */
    --primary-dark: #047857;         /* emerald-700 */
    --secondary-color: #10b981;      /* emerald-500 */
    --accent-color: #f59e0b;         /* amber-500 */
    --text-dark: #064e3b;            /* emerald-900 */
    --text-light: #374151;           /* gray-700 */
    --bg-gradient: linear-gradient(135deg, #ecfdf5 0%, #d1fae5 50%, #a7f3d0 100%);
    --glass-bg: rgba(255, 255, 255, 0.25);
    --glass-border: rgba(255, 255, 255, 0.18);
}
```

### Design Philosophy
- **Green tones**: Represent freshness and natural ingredients
- **Gold/amber accents**: Suggest warmth and traditional cooking
- **High contrast**: Ensures excellent readability
- **Premium feel**: Appropriate for traditional Dhivehi restaurant

## Implementation Guide

### How to Apply New Color Schemes

1. **Backup Current Colors**: Save the current color variables before making changes
2. **Update CSS Variables**: Replace the `:root` variables in `opening-soon.blade.php`
3. **Test Responsiveness**: Ensure colors work across all device sizes
4. **Check Contrast**: Verify text readability meets accessibility standards
5. **Update Logo Colors**: Adjust logo gradients to match new scheme if needed

### Files to Update
- `resources/views/opening-soon.blade.php` (lines 33-43)
- `resources/css/app.css` (if using CSS variables)
- Logo SVG gradients (if updating logo colors)

### Testing Checklist
- [ ] Light mode appearance
- [ ] Dark mode appearance (if applicable)
- [ ] Mobile responsiveness
- [ ] Text contrast ratios
- [ ] Brand consistency
- [ ] Logo compatibility

## Color Psychology for Restaurants

### Green Tones
- **Freshness**: Natural, healthy ingredients
- **Growth**: Prosperity and abundance
- **Calm**: Relaxing dining atmosphere

### Earth Tones
- **Warmth**: Comfortable, inviting space
- **Authenticity**: Traditional, genuine flavors
- **Stability**: Trustworthy, established business

### Gold/Amber Tones
- **Luxury**: Premium, high-quality experience
- **Warmth**: Comfort and hospitality
- **Success**: Prosperity and achievement

### Blue Tones
- **Trust**: Reliable, professional service
- **Calm**: Peaceful dining environment
- **Freshness**: Clean, modern approach

## Accessibility Considerations

### Contrast Ratios
- **Normal text**: Minimum 4.5:1 contrast ratio
- **Large text**: Minimum 3:1 contrast ratio
- **UI elements**: Minimum 3:1 contrast ratio

### Color Blindness
- Avoid relying solely on color to convey information
- Use patterns, icons, or text labels as alternatives
- Test with color blindness simulators

### Testing Tools
- WebAIM Contrast Checker
- Color Oracle (color blindness simulator)
- Browser developer tools accessibility features

## Brand Guidelines

### Logo Compatibility
All recommended color schemes are designed to work with the existing Bake & Grill logo, which features:
- Chef's hat and grill elements
- Bread and steam details
- Circular design with gradients

### Typography
- **Primary Font**: Playfair Display (elegant, traditional)
- **Secondary Font**: Inter (modern, readable)
- **Dhivehi Font**: A_Faruwa (cultural authenticity)

### Visual Hierarchy
1. **Primary**: Main brand elements, CTAs
2. **Secondary**: Supporting elements, navigation
3. **Accent**: Highlights, special features
4. **Text**: Content, descriptions
